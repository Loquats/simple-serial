# simple-serial
npm serialport to TI Launchpad MSP430

## Install
1. Install npm
2. Install serialport: `npm i serialport`
3. Load `SerialButton.ino` onto microcontroller
4. Run serial.js: `node -i -e "$(< serial.js)"`
